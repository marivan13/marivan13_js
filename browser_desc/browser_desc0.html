Mozilla Firefox

Mozilla Firefox (known simply as Firefox) is a free and open-source[17] web browser developed for Windows, OS X, and Linux, with a mobile version for Android, by the Mozilla Foundation and its subsidiary, the Mozilla Corporation. Firefox uses the Gecko layout engine to render web pages, which implements current and anticipated web standards.

Developer(s):Mozilla Foundation

Engines:Gecko, SpiderMonkey

Operating system:Windows, Unix, Android

Safari

Safari is a web browser developed by Apple Inc. and included with the OS X and iOS operating systems. First released as a public beta on January 7, 2003,[3] on the company's OS X operating system, it became Apple's default browser beginning with Mac OS X v10.3 "Panther". Safari is also the native browser for iOS.

Developer(s):Apple Inc.

Engines:WebKit, Nitro

Operating system:OS X, iOS, Windows

Internet Explorer

Internet Explorer (formerly Microsoft Internet Explorer and Windows Internet Explorer, commonly abbreviated IE or MSIE) is a series of graphical web browsers developed by Microsoft and included as part of the Microsoft Windows line of operating systems, starting in 1995. It was first released as part of the add-on package Plus! for Windows 95 that year. Later versions were available as free downloads, or in service packs, and included in the Original Equipment Manufacturer (OEM) service releases of Windows 95 and later versions of Windows.

Developer(s):Microsoft

Engines:Trident, Chakra

Operating system:Windows, Mac OS X, Solaris OS, HP-UX

Netscape Navigator

Netscape was the general name for a series of web browsers originally produced by Netscape Communications Corporation, now a subsidiary of AOL. The original browser was once the dominant browser in terms of usage share, but as a result of the first browser war, it lost virtually all of its share to Internet Explorer.

Developer(s):Netscape Communications AOL

Engines:Gecko

Operating system:Windows, Unix

Google Chrome

Google Chrome is a freeware web browser[9] developed by Google. It used the WebKit layout engine until version 27 and, with the exception of its iOS releases, from version 28 and beyond uses the WebKit fork Blink.[10][11][12] It was first released as a beta version for Microsoft Windows on September 2, 2008, and as a stable public release on December 11, 2008.

Developer(s):Google Inc.

Engines:Blink (WebKit on iOS), V8

Operating system:Windows, Unix, Android, iOS
